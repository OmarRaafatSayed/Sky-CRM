# إدارة البيانات المحفوظة - Data Management

## نظرة عامة

تم إنشاء مجموعة من المكونات لإدارة البيانات المحفوظة محلياً في المتصفح ومسحها لرؤية آخر التحديثات من الخادم.

## المكونات المتاحة

### 1. DataCleaner
مكون بسيط لمسح جميع البيانات المحفوظة محلياً وإعادة تشغيل التطبيق.

**الميزات:**
- مسح localStorage
- مسح sessionStorage  
- مسح IndexedDB
- تسجيل الخروج من الجلسة الحالية
- إعادة تحميل الصفحة تلقائياً

**الاستخدام:**
```tsx
import { DataCleaner } from '@/components/DataCleaner';

<DataCleaner />
```

### 2. StorageManager
مكون متقدم لإدارة البيانات المخزنة مع إمكانية عرض أحجام البيانات ومسح كل نوع بشكل منفصل.

**الميزات:**
- عرض حجم البيانات في localStorage
- عرض حجم البيانات في sessionStorage
- تقدير حجم IndexedDB
- مسح كل نوع بيانات بشكل منفصل
- عرض الحجم الإجمالي للبيانات

**الاستخدام:**
```tsx
import { StorageManager } from '@/components/StorageManager';

<StorageManager />
```

### 3. OrganizationSelector
صفحة لاختيار المنظمة مع خيارات إدارة البيانات.

**الميزات:**
- عرض المنظمات المتاحة للمستخدم
- اختيار منظمة للدخول إليها
- خيارات مسح البيانات المحفوظة
- واجهة سهلة الاستخدام

**الاستخدام:**
```tsx
import { OrganizationSelector } from '@/components/OrganizationSelector';

<OrganizationSelector />
```

## التكامل مع التطبيق

### في صفحة المصادقة (AuthPage)
تم إضافة مكون DataCleaner في نهاية صفحة تسجيل الدخول لإتاحة مسح البيانات قبل تسجيل الدخول.

### في App.tsx
تم إضافة منطق لعرض OrganizationSelector عندما يكون المستخدم مسجل دخول لكن لا يملك منظمة محددة.

### Route جديد
تم إضافة route `/select-organization` لصفحة اختيار المنظمة.

## حالات الاستخدام

1. **مشاكل في تسجيل الدخول**: استخدم DataCleaner لمسح البيانات المحفوظة
2. **عدم ظهور آخر التحديثات**: امسح البيانات لتحديث المحتوى من الخادم
3. **مشاكل في اختيار المنظمة**: استخدم StorageManager لإدارة البيانات بشكل دقيق
4. **تنظيف دوري**: استخدم StorageManager لمراقبة استهلاك التخزين

## الأمان

- جميع العمليات تتم محلياً في المتصفح
- لا يتم إرسال أي بيانات للخادم أثناء عملية المسح
- يتم تسجيل الخروج تلقائياً لضمان الأمان
- إعادة تحميل الصفحة تضمن تحديث جميع البيانات

## ملاحظات تقنية

- يستخدم Supabase للمصادقة وإدارة الجلسات
- يدعم جميع أنواع التخزين الحديثة في المتصفحات

- رسائل واضحة باللغة العربية للمستخدم
